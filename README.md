# UMPL Killer bot
This bot was created for the game "Killer" in the Ugra Physics and Mathematics Lyceum.

## Deploy
1. Set env variables:
    * TOKEN - Telegram bot token
    * DB_FILE - DB path
    * ADMIN_ID - Telegram id admin
2.  Launch

      Local
      
      ```
      pip install -r requirements.txt
      python main.py
      ```
      Docker
      ```
      docker-compose up
      ```
